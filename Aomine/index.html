🔔 Material Theme para VSCode.

📌 Hoy les comparto el tema de color e iconos que yo uso y como cambiar la fuente de letra en VSCode.
✔ Tema: Material Theme
Un tema que cuenta con distintos y agradables colores tanto de fondo como de letras.
✔ Iconos: Material Icon Theme
Un pack de iconos muy variados no solo para lenguajes sino también para algunas carpetas con nombres comunes como el famoso (model, view y controller).
✔ Fuente de letra: Comic Sans MS
Un fuente de letra que me agrada mucho, en este punto tu puedes elegir la fuente de letra que mas te guste aunque algunos desarrolladores recomiendan no hacerlo y dejarlo en la fuente de letra por defecto(consolas) pero creo que es algo que depende de cada persona.

🔔 Sublime Text 3.2 fue lanzado hoy !!! 🔔

⚠ Para evitar discusiones innecesarias aclaro que VSCode es actualmente el mejor editor de código y no lo digo yo, lo dicen las estadísticas.
📣 Su principal nueva característica es la itegración con GIT que se complementa con un gestor visual de repositorios llamado sublime merge.
✔ Sublime Text es un editor de código enfocado en tecnologías web desarrollado en phyton, por eso es muy rápido y gracias a la implementación de algunos plugins se puede potenciar mucho más. pero frente a otros editores como vscode carece de muchas cosas indispensables para el desarrollo actual como una terminal integrada, otra era la integración con git que gracias a esta actualización ya lo tiene.
🌀 Sublime fue considerado unos de los mejores editores o el mejor por algunos desarrolladores por mucho tiempo hasta la llegada de vscode que cuenta con cosas increíbles y rápidamente se convirtió en el editor favorito de la gran mayoría de desarrolladores y considerado el mejor por muchos otros(yo incluido) y que poco a poco se parece más a un IDE y lamentablemente sublime no implemento nuevas características y se quedo atrás, pero gracias a esta actualización se esta volviendo a hablar de este editor. ¿algún día implementara una terminal integrada propia? y ya se que algunos plugins te perminten tenerla, volviendo al punto ¿sublime text intenta revivir y poder estar a la altura de vscode? espero que sea así, ya que esto genera competencia y hace que los desarrolladores de estos se esfuercen más para entregar un mejor producto y los beneficiados somos nosotros que tendremos mejores editores, y espero que vscode no monopolice el mercado.
